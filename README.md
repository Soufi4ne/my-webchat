# My-webchat

## Introduction

> Just a simple project to train with socket.io

## Code Samples

Use this in your package.json (after you do the installation instruction)
> "scripts": {
    "start": "node server",
    "dev":"nodemon server"
  },

## Installation

- npm init
- npm install express
- npm install socket.io
- npm moment
- npm install -D nodemon
